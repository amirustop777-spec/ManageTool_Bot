# 📊 ManageTool Bot

Персональный Telegram-бот для мониторинга курсов валют и финансовой аналитики. 
Проект разработан для автоматизации расчетов и получения актуальных данных из облачных ИИ-моделей.

## 🚀 Основные возможности
* **Конвертация валют**: Автоматический расчет курсов через API ЦБ РФ.
* **ИИ-Аналитика**: Генерация финансовых отчетов (USD, EUR, KZT) с использованием FreeGPT (G4F).
* **Комиссия**: Автоматический учет маржи (2%) для удобства ведения учета.
* **Асинхронность**: Быстрая работа благодаря библиотекам `aiogram` и `aiohttp`.

## 🛠 Технологии
- **Язык**: Python 3.10+
- **Движок бота**: Aiogram 3.x
- **ИИ-модуль**: G4F (GPT-4o mini)

# 🤖 ManageTool

Персональный финансовый Telegram-бот с функциями конвертации валют, курсов ЦБ РФ и AI-аналитикой.

## 📋 Функционал

| Команда | Описание |
|---------|---------|
| `/start` | Запуск бота |
| `Курсы валют` | Показать курсы USD, EUR, CNY, KZT |
| `Калькулятор` | Конвертировать валюту в рубли |
| `Прогноз от ИИ` | AI-анализ валюты |
| `FAQ` | Часто задаваемые вопросы |
| `🔙Cancel` | Отмена и возврат в меню |

## 🛠 Установка

```bash
pip install aiogram aiohttp g4f aiohttp_socks
```

## ⚙️ Настройка

Отредактируйте `config.py`:

```python
TOKEN = 'YOUR_TELEGRAM_BOT_TOKEN'
PROXY_URL = 'socks5://IP:PORT'  # опционально
KEY = 'YOUR_HUGGINGFACE_KEY'    # для AI
```

## 🚀 Запуск

```bash
python run.py
```

## 📊 Источники

- Курсы валют: [ЦБ РФ](https://www.cbr-xml-daily.ru/)
- AI: g4f (GPT-4o mini)
