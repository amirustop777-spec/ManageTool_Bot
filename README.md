# 📊 ManageTool Bot

Персональный Telegram-бот для мониторинга курсов валют и финансовой аналитики. 
Проект разработан для автоматизации расчетов и получения актуальных данных из облачных ИИ-моделей.

## 🚀 Основные возможности
* **Конвертация валют**: Автоматический расчет курсов через API ЦБ РФ.
* **ИИ-Аналитика**: Генерация финансовых отчетов (USD, EUR, KZT) с использованием FreeGPT (G4F).
* **Комиссия**: Автоматический учет маржи (2%) для удобства ведения учета.
* **Асинхронность**: Быстрая работа благодаря библиотекам `aiogram` и `aiohttp`.

## 🛠 Технологии
- **Язык**: Python 3.10+
- **Движок бота**: Aiogram 3.x
- **ИИ-модуль**: G4F (GPT-4o mini)
- **База данных**: (В разработке)
